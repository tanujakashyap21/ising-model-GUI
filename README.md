# 2D-Ising-Model-Monte-Carlo-GUI

Monte Carlo simulation of the two-dimensional Ising model with an interactive graphical user interface (GUI) for studying magnetic phase transitions and thermodynamic properties.

## Overview

This project implements a Monte Carlo simulation of the two-dimensional Ising model using the Metropolis algorithm. An interactive GUI was developed to visualize spin configurations and analyze thermodynamic quantities.

## Objectives

- Simulate the two-dimensional Ising model
- Study magnetic phase transitions
- Compute thermodynamic observables
- Visualize spin lattice evolution
- Develop an interactive GUI for simulation and analysis

## Physical Quantities Investigated

- Magnetization
- Magnetic Susceptibility
- Heat Capacity
- Internal Energy
- Critical Temperature Behavior

## Methodology

1. Initialize a two-dimensional spin lattice.
2. Apply the Metropolis Monte Carlo algorithm.
3. Calculate thermodynamic observables at different temperatures.
4. Analyze phase transition behavior.
5. Visualize lattice evolution and simulation results through a GUI.
